# Sky Bounce 🎮

My first 2D game made with Unity.

This project was created as a practice to learn 2D game development.
The game idea is inspired by Flappy Bird, and I made it for my niece because she wanted a simple and fun game.

## Features
- Simple 2D gameplay
- Score system
- Sound effects
- Easy controls

## Tools Used
- Unity
- C#
- Canva (for some design elements)

## What I Learned
- 2D game mechanics
- Collision handling
- UI basics in Unity
- Scene management

## Note
This is my first 2D game project, and it helped me improve my game development skills.

⚠️ This project is still a work in progress, and I am continuing to improve and update it.
